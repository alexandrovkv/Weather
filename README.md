# Weather-App
Weather App - display current weather information
### Supported weather data providers
* [Open Weather Map](http://openweathermap.org)
* [World Meteorological Organization](https://www.wmo.int)
* [Weather Underground](http://www.wunderground.com)
* [Dark Sky](https://darksky.net/poweredby/)
* [Weather Bit](https://www.weatherbit.io/)
* [World Weather Online](https://www.worldweatheronline.com)
